*Antes de iniciar el instalador copie el código fuente del proyecto dentro de:
   ./Programa/QtRatWalk, que es todo el proyecto de github a excepción de la
   carpeta "Install"
*Para instalar abra una terminal en este directorio y ejecute el siguiente comando:
   "sh install_ratwalk.sh"
*Una vez terminada la instalación debería encontrar en su escritorio un archivo
 ejecutable llamado "QtRatWalk" y una carpeta llamada "resources".
